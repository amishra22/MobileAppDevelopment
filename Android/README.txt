#List of Apps

1) LifeCycle : This app is a demonstration of a simple 2 activity app in android. When a user clicks on a button from the main activity a dialog box open (2nd Activity).

2) MovieDatabase : This app is a demonstration of a list view in android app development which show different movies stored in different genre section. Once you click the genre it lists all the movie present in that genre and if you click the movie it lists its details.
